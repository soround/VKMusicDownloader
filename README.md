# **Установка**
1.  Установить [python 3.8+](https://www.python.org/) и выше 
2.  Скачать проект
3.  Перейти в папку с проектом `cd VKMusicDownloader`
4.  Установить необходимые модули
    ```console 
    pip install -r requirements.txt
    ```
5. Запустить `main.pyw`
